# oversync-links

Link resolver with WASM plugin support for oversync.

Part of [oversync](https://github.com/overrealdb/oversync).

## What this crate provides

- **Cross-source entity resolution** -- match rows from different origins that refer to the same entity
- **Built-in match rules** -- planned name/key-based matching strategies
- **WASM plugin support** -- pluggable matching logic via WebAssembly modules

This crate is a stub. Implementation arrives in Phase 5 of the oversync roadmap.
